# 🔌 HANZZO NETWORK PORT PRO v1.5.5

**Профессиональный мониторинг сетевых соединений для Русская Рыбалка 4**

## 📋 О программе

**HANZZO NETWORK PORT PRO** — это профессиональный инструмент для мониторинга сетевых соединений игры **Русская Рыбалка 4**. Программа в реальном времени отслеживает параметры подключения к игровым серверам, предоставляя детальную статистику и визуализацию.

### ✨ Основные возможности

| Функция | Описание |
|---------|----------|
| 🎮 **Автоматическое обнаружение** | Определяет активное соединение игры в реальном времени |
| 📊 **Мониторинг пинга** | Измерение задержки с точностью до 1 мс |
| 🌍 **Геолокация сервера** | Определение страны и города сервера |
| 📈 **Визуализация** | График изменения пинга в реальном времени |
| 🔥 **Firewall Manager** | Управление диапазонами портов для оптимизации соединения |
| ⭐ **Избранные порты** | Сохранение и быстрый доступ к важным портам |
| 📜 **История портов** | Отслеживание сессий подключений |
| 📸 **Скриншоты** | Быстрое создание и сохранение скриншотов интерфейса |
| 🖥️ **Мини-HUD** | Компактное окно с пингом при сворачивании |
| 🔔 **Уведомления** | Системные уведомления о важных событиях |

## 🚀 Установка

### Системные требования

- **ОС:** Windows 10 / 11 (64-bit)
- **Процессор:** 1 GHz или выше
- **ОЗУ:** 256 MB
- **Место на диске:** 50 MB
- **Права администратора** (для работы с брандмауэром)

### Установка из исполняемого файла

1. Скачайте последнюю версию из [релизов](https://github.com/hanzzo/rf4-port-monitor/releases)
2. Запустите `HANZZO_Network_Port_Pro_Setup.exe`
3. Следуйте инструкциям установщика
4. После установки запустите программу от имени администратора

### Портативная версия

1. Распакуйте архив `HANZZO_Network_Port_Pro_Portable.zip`
2. Запустите `HANZZO_Network_Port_Pro.exe` от имени администратора

## 🎯 Использование

### Быстрый старт

1. **Запустите программу** от имени администратора
2. **Запустите игру** Русская Рыбалка 4
3. Программа **автоматически обнаружит** соединение
4. Начнётся **мониторинг пинга** в реальном времени

### Горячие клавиши

| Комбинация | Действие |
|------------|----------|
| `Ctrl + W` | Свернуть/развернуть окно |
| `Ctrl + F` | Открыть меню функций |
| `Ctrl + P` | Сделать скриншот |
| `Ctrl + I` | Информация о безопасности |
| `Ctrl + Q` | Закрыть программу |
| `Ctrl + Shift + F` | Открыть Firewall Manager |
| `Ctrl + U` | Проверить обновления |
| `Ctrl + ,` | Открыть настройки |
| `Ctrl + H` | История портов |

### Управление диапазонами портов (Firewall Manager)

1. Нажмите `Ctrl + Shift + F` или выберите в меню "Выбор диапазонов"
2. Выберите один из 5 предустановленных диапазонов портов
3. Нажмите **"ПРИМЕНИТЬ ПРАВИЛО"**
4. **Перезапустите игру** для применения изменений

### Избранные порты

- **Добавление:** Нажмите на звездочку ☆ рядом с номером порта
- **Управление:** Правый клик по звездочке → контекстное меню
- **Быстрый доступ:** Избранные порты отображаются в контекстном меню

## 🛡️ Декларация безопасности

Программа **НЕ является** модом, читом или взломом. Она:

- ✅ Работает только в **режиме чтения** сетевых данных
- ✅ **Не внедряется** в процессы игр
- ✅ **Не модифицирует** системные файлы
- ✅ **Не изменяет** сетевые пакеты
- ✅ **Не нарушает** правила использования ПО

## 🐛 Устранение неполадок

### Программа не видит игру

1. Убедитесь, что игра **Русская Рыбалка 4** запущена
2. Запустите программу **от имени администратора**
3. Проверьте активное сетевое соединение

### Не применяются правила брандмауэра

1. **Перезапустите программу** от имени администратора
2. Убедитесь, что Windows Defender не блокирует программу
3. Временно отключите антивирус

### Мини-HUD не появляется

1. Проверьте настройки: **Настройки → Система → Показывать мини-HUD**
2. Убедитесь, что функция включена
